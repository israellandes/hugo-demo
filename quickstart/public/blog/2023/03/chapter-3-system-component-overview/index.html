Chapter 3 - System Component Overview

2023-03-14 723 words 4 mins read

CompTIA A+

This article is Chapter 3 in the CompTIA A+ certification course on Alison.com that teaches a System Component Overview of computers.

Processors and CPUs

Processors and the Operating systems are the Work horse of the computer.

• Processors and CPUs

  • Brains in the computer
  • Allow it to perform many tasks really fast
  • Processor performs calculations based on request from Operating System, which controls acess to system memory

• Processing speeds

  • Speed or power of a processor
  • Calculated in Megahertz, Gigahertz or Millions of instructions per second

• Operating systems and controls

  • Control the tasks that a computer carries out.
  • Manages software and hardware resources of computer

This diagram shows how data input flows from a processor to produce an output that is then stored in memory.

Many ancillary or supportive devices that help make these process happen such as:

• Input Devices

  • Card readers
  • Keyboard
  • Mouse

• Output Devices

  • Monitor
  • Printer

• Storage Devices

  • Hard Drives
  • RAM
  • Tapes

Power Supplies

• Power supply voltage requirments

  • Convert 120V of alternating current from electrical outlet and convert to nessisary voltage for computer.

Each device requires diffrent type of voltage requirment.

• 3.3V

  • Less common, found internally in computer

• 5V

  • Less common, found internally in computer

• 12V

  • Most common

Storage Devices

Storage Devices 1. Storgae Devices 2.

• Hard drives

  • Most common

• Magnetic tapes

  • Used for archiving data

• Flash drives

  • Thumbdrives, zip-drives for quick memory transfor data

• Solid-state drives

  • Use a flash type memory to store data.
  • Computers memory is solid state storage medium, like random acess memory.
  • Beneficial because there no moving parts, less worn out.

Connecting hard dives to the motherboard can be done using one of these technologies.

  • IDE - Integrated Drive Electronics
  • SATA - Serial ATA
  • SCSI - Small Computer System Interface

Hard drives are usually manufactured with one of these technologies which are then connected through bus mechenisms called a Periferial Component Interconnect.

Adapter Cards

Allow you to adapt computer to perform functions such as display a picture, or connect to a network. They are integrated into motherboard of computer.

• Adapter cards

  • Mouse controllers
  • Serial & Paralel Ports
  • USB Ports
  • Displays

• ISA, AGP, PCI, PCI Express and others

  • Termonoligy used to describe adaptor slots

• Video Cards

  • Used to produce pictures
  • Measured using bits and colors ex: 16Bit or 65K Color
  • Resolution measured by Pixles ex: 1280 x 1024

BIOS/CMOS

BIOS is responsible for controling and managing system processes such as system boot and interaction with other components on motherboard.

• BIOS

  • Basic Input Output System
  • Software stored on read only memory chip on motherboard

• CMOS

  • Complementary Metal Oxcide Semiconductor
  • Stores hardware configuration settings between reboots
  • Supplied with a batteryon motherboard
  • Stores settings related to BIOS

• UEFI

  • Unified Extensible Firmware Interface
  • Low level code to control motherboard procesess
  • Also includes diognostic tools that support network booting and commands.

• Storage Settings

  • Stored in RAM memory and require power to maintain information
  • Some settings include:
    • Which drives are present?
    • How much memory is installed?
    • What type ofmouse is installed?
    • Any required resources.
    • Date & time.
    • Power on password.

Chips

Computer Chips are integrated circuts that contain transistors that perform a function. They make a decision or actually create a function.

• Chipsets

  • Used to manage data flow between processor and memory,as well as devices.

• Firmware

  • Fills in between software and hardware
  • A program in code, stored on chip that provides low level controls for hardware that help you communicate.
  • Faccilitates functioning of the integrated chip circuits which add card functionality.

Input and Output Devices

Printer Monitor

Most common Input and Output devices.

• Printers

  • Allow users to produce paper output from computers

• Monitors

  • Allow users to produce images and video displayes.

Others include scanners, faxes or cameras.

Ports

• Ports

  • Serve as connection points for cables which allow data between a computer and device.

• Serial

  • Used to connect devices like modems

• Parallel

  • Connections for printers

• Keyboard and mouse

  • specific port connectiosn which enable Keyboard and mouse.

• USB

  • Universal Serial Bus
  • Can connect a number of diffrent devices such as mouses, keyboards, printers, scanners, modems, flashdrives, and so on.

• Video

  • Used for plug and play devices to produce picture outputs.
